# Sign Language Recognition System

## Features
- Hand gesture recognition using MediaPipe
- Gesture to speech conversion
- Voice to gesture mapping
- Real-time camera processing
- Gesture image overlay

## Controls
- Press `v` → Speak input
- Show hand → Auto voice output
- Press `q` → Quit

## Requirements
- Python 3.10
- OpenCV
- MediaPipe
- pyttsx3
- SpeechRecognition
